# 桃策官网上线检查清单

## 必查项

- [ ] 商务邮箱已替换为真实可用邮箱
- [ ] 微信号已替换为真实可联系账号
- [ ] 首页表单提交后能正常打开邮件客户端
- [ ] `sitemap.xml` 中的网址已改成当前实际域名
- [ ] `robots.txt` 中的网址已改成当前实际域名
- [ ] 没有公开真实手机号，避免骚扰电话
- [ ] 页脚公司主体仍保留“以工商登记核准为准”或已改成营业执照名称
- [ ] 手机端打开首页，导航栏 Logo 不遮挡菜单
- [ ] 首页、博客页、404 页都能正常打开
- [ ] favicon 能在浏览器标签中显示

## 暂时没有正式域名时

- [ ] 不要启用 `CNAME`
- [ ] 可以先用 GitHub Pages 临时域名测试
- [ ] 等正式域名确定后再提交 sitemap

## 有正式域名时

- [ ] `CNAME.example` 改名为 `CNAME`
- [ ] `CNAME` 文件内填写正式域名
- [ ] GitHub Pages 设置里添加自定义域名
- [ ] 域名 DNS 已解析
- [ ] HTTPS 已开启
